# Mates con Fiore — Landing Page

Proyecto de portafolio: landing page para clases particulares de matemáticas.

## Datos del servicio

| Campo | Detalle |
|---|---|
| Nombre | Mates con Fiore |
| Tipo | Clases particulares de matemáticas |
| Niveles | Secundaria, adultos / todas las edades |
| Modalidad | Presencial en su casa, online por videollamada, grupos reducidos |
| Ciudad | Gobernador Gálvez, Santa Fe |

## Estructura del sitio

```
index.html
├── Nav          → Logo + links + botón "Reservar clase"
├── Hero         → Frase principal + botón CTA
├── Servicios    → 3 tarjetas (secundaria, adultos, grupos)
├── Metodología  → Cómo trabaja Fiore (3 pasos simples)
├── Sobre mí     → Quién es Fiore, su experiencia
├── Contacto     → Formulario + WhatsApp + modalidades
└── Footer       → Horarios, redes, © 2025
```

## Stack técnico

- HTML5 semántico
- CSS3 (variables, Flexbox, Grid, media queries)
- JavaScript vanilla (validación de formulario, menú hamburguesa, scroll suave)
- Deploy: Netlify (gratis)
